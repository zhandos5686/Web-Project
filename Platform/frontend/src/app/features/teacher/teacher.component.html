How content creation works

  1. Create a course.
  2. Create a module inside that course.
  3. Create a lesson inside that module.
  4. Create one quiz and one written task for the lesson.
  5. Add questions and choices to the quiz.

Quiz belongs to a lesson, not directly to a course. A course contains modules, and modules contain lessons.

The forms below use only content owned by the logged-in teacher.
