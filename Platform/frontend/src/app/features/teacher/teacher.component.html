Content creation

Build course structure first, then add quizzes, choices, and written tasks to lessons.

Creation order

  1. Create a course.
  2. Add modules to that course.
  3. Add lessons to modules.
  4. Add one quiz and one written task to a lesson.
  5. Add questions and choices to the quiz.

All dropdowns use only content owned by the logged-in teacher.
