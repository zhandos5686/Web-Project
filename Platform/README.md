# English Learning Platform

A full-stack English learning platform built with Angular and Django REST Framework.

## Goal
Build a strong and clean base version first, then extend it with advanced features like AI assistant, translation tools, irregular verbs trainer, and personalized learning support.

## Planned Base Features
- Authentication
- Student and teacher roles
- Course catalog
- Course detail page
- Modules and lessons
- YouTube video lessons
- Quizzes and tasks
- Progress tracking
- My courses
- My tasks
- Profile
- Teacher dashboard
- Booking live lesson slots

## Development Strategy
1. Build clean base version
2. Test every block
3. Understand every part of the code
4. Add advanced features later
